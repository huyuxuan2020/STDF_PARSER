### ✨ 新功能

- **概览页 CP/FT 一键切换**：自动识别不准时，直接点右上角标签在 CP ↔ FT 之间切换，关键字段跟着更新。
- **测试项右键跳源记录**：在测试项表的值单元格右键，即可跳到产生该值的 PTR/MPR/FTR 记录；菜单里也保留了「复制值」。
- **字段详情标注继承值**：PTR/MPR 记录里被省略但从首条继承的字段（RES_SCAL、UNITS、LO/HI_LIMIT 等）会显示继承来的原始值 + 蓝色「继承」徽章。
- **搜索进度条**：全文搜索时显示带百分比的进度条 + 已扫描 / 总记录数，不再只有一个转圈图标。
- **启动检查更新只显示红点**：启动时后台静默检查到新版本时，不再自动弹窗，仅在左侧「更新」按钮右上角显示红点，点了才展示详情。
- **跨多版本更新说明合并显示**：从旧版本升级时，弹窗按版本倒序拼接展示中间所有版本的更新说明，不会漏看。
- **弹窗按 Markdown 渲染**：更新说明弹窗按 Markdown 排版（标题、列表、加粗、行内代码），不再是一坨源码。

### 🐛 Bug 修复

- **大文件搜索卡死**：全文搜索时 UI 主线程被阻塞，鼠标 beach ball 且没有反馈；搜索现在后台运行、进度实时更新。
- **测试项 / 记录页打开慢**：测试项、记录等重后端命令改为在后台任务里运行，切换页签时鼠标不再卡顿，期间其他页面照常可用；加载文案也更清晰。
- **首次启动手动检查更新时弹窗一闪即消**：静默检查与手动点击竞态导致弹窗被覆盖。
- **离线时更新按钮显示技术错误**：识别到网络问题时改为友好提示，并说明离线不影响 STDF 解析等本地功能。
- **发版竞态**：安装包尚未上传完成、`latest.json` 却已可见的窗口期消除；现在全部平台构建完才正式发布，检测到更新即可正常下载。
